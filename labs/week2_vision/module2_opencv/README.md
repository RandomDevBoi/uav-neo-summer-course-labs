# Week2 Vision — Module2 Opencv

Thresholding and morphology (erode/dilate, blur, Sobel) on the live downward camera.

## Simulator lab

```bash
drone open_sim                 # launch the Unity sim once
drone sim week2_vision/module2_opencv/main.py            # run all steps (student)
drone sim week2_vision/module2_opencv/main_solution.py   # reference flight
drone sim week2_vision/module2_opencv/tasks/<step>.py    # run a single step
```

Steps:

1. `step1_threshold.py`
2. `step2_morphology.py`
3. `step3_blur_edges.py`

Student stubs live in `tasks/`; completed references in `solutions/`.
