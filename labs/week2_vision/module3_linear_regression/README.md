# Week2 Vision — Module3 Linear Regression

Detect a colored ground line, fit it with least-squares regression, then follow it.

## Simulator lab

```bash
drone open_sim                 # launch the Unity sim once
drone sim week2_vision/module3_linear_regression/main.py            # run all steps (student)
drone sim week2_vision/module3_linear_regression/main_solution.py   # reference flight
drone sim week2_vision/module3_linear_regression/tasks/<step>.py    # run a single step
```

Steps:

1. `step1_detect_line.py`
2. `step2_fit_line.py`
3. `step3_follow_line.py`

Student stubs live in `tasks/`; completed references in `solutions/`.
